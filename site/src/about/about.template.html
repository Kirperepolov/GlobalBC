About this site

How it was started

While learning how to use AngularJS I decided to gather all the previously passed materials on JavaScript in one place with easy access to. For this I started my own Angular project with just two views:

  • the list of all the sessions I passed in GlobalLogic JS Basecamp;
  • the links to all the learning and practicing materials regarding this sessions.

Foreasmuch, I don't have my web-server with built-in database yet, I started preparing a JSON file with all the needed information manually. It's a long way to the top if you create jsons manually, so I just filled in the first session information and decided to create an admin view with a form for further information filling.

After the preliminary site styling I've got an idea that it could be a nice pet-project with appropriate information about me, links to my other works, links to my profiles in social networks and github and, ofcourse, my contact information.

Later than another idea came to me: I can to move some data to a free web-server, i.e. heroku, and manage it with my admin form. I believe I can handle it, moreover, I have an example of Yaakov Chaikin's server application on ruby to start with.
